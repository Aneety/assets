# Aneety logo

Fonte canônica dos assets reutilizáveis da marca Aneety.

## Arquivos

- `aneety-mark.svg` - símbolo quadrado para avatar, favicon e ícone.
- `aneety-logo.svg` - logo horizontal com símbolo e wordmark.
- `social-preview.svg` - fonte vetorial do social preview do repositório.
- `social-preview.png` - imagem pronta para upload no GitHub Social preview, 1280x640 px.
- `social-preview.jpg` - alternativa compacta para upload no GitHub Social preview, 1280x640 px.

## Regra

Todo asset reutilizável deve preservar uma versão SVG neste repositório. Derivações PNG/JPG podem existir apenas como exportações prontas para uso.
