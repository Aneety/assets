# Aneety assets

Repositório canônico de assets reutilizáveis da organização Aneety.

## Logo

Assets iniciais em [`logo/`](logo/):

- marca Aneety em SVG;
- logo horizontal em SVG;
- social preview em SVG, PNG e JPG no tamanho recomendado pelo GitHub: 1280x640 px.

## Política

Todos os assets reutilizáveis devem ter fonte SVG versionada neste repositório. Repositórios de implementação devem consumir ou referenciar estes arquivos para evitar variações divergentes.
